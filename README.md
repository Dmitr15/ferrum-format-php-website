### Simple responsive website on HTML/CSS/PHP and BOOTSTRAP with products filter

### Original site: https://ferrumformat.com/catalog/sekcionnye-pochtovye-yaschiki?block=premium.1

#### Frontend: HTML/CSS and BOOTSTRAP framework
#### Backend: pure php 7
#### Data Base: MySql via phpMySqlAdmin

### Implemented functionality:
#### 1)Responsive design for devices of any screen size.
#### 2)Filter products on the page by price, name, description, brand.
#### 3)Save selected filters when sending a request to the server.
